# img2eink

```
Usage: img2eink [OPTIONS] --width <WIDTH> --height <HEIGHT> <TARGET>

Arguments:
  <TARGET>  File or directory to convert

Options:
      --help             Show this help
  -r, --recurse          Traverse the directory recursively
  -d, --dest <DEST>      File or directory to write converted image to
  -w, --width <WIDTH>    Screen width of your E-ink device
  -h, --height <HEIGHT>  Screen height of your E-ink device
```
